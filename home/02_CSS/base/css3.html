Блок навигации
  • Ширина: 15em
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации дополните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере
Статья 1 Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех статьях "статьях" сделайте отступ снизу от соседей равным .2em, при помощи свойства margim-bottom
Статья 2 Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех статьях "статьях" сделайте отступ снизу от соседей равным .2em, при помощи свойства margim-bottom. Картинка вставлена при помощи тега, плавание организовано по правому краю внутри данной статьи для этой картинки.
Нижнний блок, подвал. Занимает всю доступную ширину. Высота определяется помещённым внутрь контейнером
